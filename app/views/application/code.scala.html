Level 1 Playback

The scope at this base level is to offer high quality video viewing through a media player.
Ayamel
h5Clip
MediaController
MediaControls
TimedMedia
UniformAspectRatio
Video
VideoPlayer
Here is a example of utilizing the code above to create a level 1 playback. 
//ResourceId identifies which video you are looking for from the database
var resourceId = "512cfb8b35e544210c000001";

$(function() {
    // Load the resource
    // This script locates the video with resourceid from the database and retrieve necessary video files
    new Resource(resourceId, function(resource) {
        // Install the html5 video player
        // Elements including volume control, progress bar, play/pause button are all installed here
        Ayamel.AddVideoPlayer(h5PlayerInstall, 1, function() {
            // Create the player
            var videoPlayer = new Ayamel.VideoPlayer({
                element: document.getElementById('player'),
                aspectRatio: 45,
                resource: resource
            });
        });
    });
});
